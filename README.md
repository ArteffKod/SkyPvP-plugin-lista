# DreamCraft SkyPvP plugin lista

A minecraft szervereknél nagyon fontos optimalizálni a szerver működését ezzel kihasználni a szervergép, szerver hoszt teljes teljesítményét
erre vannak kifejezett oldalak amik leírják pontosan mit és hogyan tegyél mint pédául [Minecraft szerver optimalizácós útmutató](https://github.com/YouHaveTrouble/minecraft-optimization), vagy ez [Optimalizáld Aikar Flags generátor](https://flags.sh/)

### Ezek mellett nagyon fontos, hogy a szerveren a megfeleő fork fusson, ami kezeli a szervert, itt egy lista a forkokról teljesítményi listába sorolva:
